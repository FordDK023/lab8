## รายละเอียดของระบบ
ระบบนี้สามารถทำงานได้ดังนี้
- แสดงรายการสินค้าทั้งหมด
- เพิ่มข้อมูลสินค้าใหม่
- แก้ไขข้อมูลสินค้า
- ลบสินค้าออกจากระบบ
- อัปโหลดรูปภาพสินค้า
- ตรวจสอบสถานะสินค้าคงเหลือ

## เครื่องมือที่ใช้
- Node.js
- Express.js
- MySQL (ใช้งานผ่าน MAMP)
- EJS Template Engine
- Bootstrap 5

## ขั้นตอนการติดตั้งและใช้งาน
1. ดาวน์โหลดโปรเจกต์นี้มาไว้ในเครื่อง หรือ clone จาก GitHub
2. เปิดโปรเจกต์ด้วยโปรแกรม VS Code
3. ติดตั้ง dependencies ที่จำเป็น โดยใช้คำสั่ง
npm install
4. เปิดโปรแกรม MAMP และกด Start Servers
5. เข้า phpMyAdmin แล้ว import ไฟล์supermarket_db.sqlเพื่อสร้างฐานข้อมูล
6. ตรวจสอบไฟล์ `app.js` ให้ตั้งค่าการเชื่อมต่อฐานข้อมูลให้ถูกต้อง เช่น
host: '127.0.0.1'
user: 'root'
password: 'root'
port: 3306
database: 'supermarket_db'
7. รันโปรเจกต์ด้วยคำสั่ง
node app.js
8. เปิดเว็บเบราว์เซอร์ แล้วเข้าใช้งานที่
http://localhost:3000

- ต้องเปิด MAMP ก่อนใช้งานทุกครั้ง
- ต้องมีไฟล์รูป `no-image.png` อยู่ในโฟลเดอร์ public/assets/img/
- ไม่ต้องอัปโหลดโฟลเดอร์ node_modules ขึ้น GitHub
